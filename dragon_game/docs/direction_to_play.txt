This game is a python game which you need to run from terminal first and after running it in terminal you'll get an url and by opening that url you'll reach at the first level of the game. You need to play the game by typing your answers in a box given below the description of each level. 

To play this Game one need to install the following package :
web.py : sudo pip install web.py

Before playing this game you need to run the command in terminal : export PYTHONPATH=$PYTHONPATH:.
After going to the directory dragon_game, you need to run the command in the terminal : python bin/app.py

Now copy and paste the url appears in the terminal in your browser and click enter. 

PLAY THE GAME NOW.
